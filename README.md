## 简介
基于utools的局域网文件共享插件

提供简单，快速，轻便，方便的文件共享

## 开发相关
main文件夹：生成main对应的应用页面

preload文件夹：生成preload.js文件

utools文件夹：main和preload的build输出路径，插件最终基于该目录运行与打包
